# Smart_Task_Planner

### Features
1. Zapier collects Assigned To and Task through a Google Form and sends the data to ClickUp 
2. Automatically maps Assigned To into the Name field and Task into the Task field  
3. Make.com monitors new or updated tasks in ClickUp and pulls the Task field data
4. Sends the task details to an AI agent, which generates a detailed step-by-step solution
5. Updates the Task Steps custom field in ClickUp, keeping everything organized with no manual input
